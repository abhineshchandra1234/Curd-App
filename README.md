# Curd-App

## Overview, Purpose & Features
- it is a simple app which will show subcribers name in recycler view.
- it is uses mvvm architecture
- user can add, edit && delete subscribers

## Concepts Used
- Navigation Component
- Recycler View
- Fragments
- Coroutines
- ViewModel && LiveData
- Room
- ViewPager2
- ViewBinding
- Dagger & Hilt
